# Snap test-seams example

Combines the code samples for the [snap.js test seams walkthrough](https://thescottyjam.github.io/snap.js/#!/seams) into a single project.

Use the js/ or ts/ folder according to your preferred language.

## Commands

First, make sure you are inside of js/ or ts/.

If in ts/, you'll need to install the TypeScript dependency, `npm ci`.

Run the js or ts project with

```sh
npm start your-todo-item-here
```

Run the tests for the project with:

```sh
npm test
```

## Requirements

Node v22.18 or later is required to run the ts/ project.
Node v18.19 is or later is required to run the js/ project.
